Программа подсчитывает количество учеников в день присутствующих на уроках и их сумму за день
Для работы необходимо установить Python3 и зависимости указанные в файле pip_requirements.txt, также необходимо скачать Chromdriver в директорию с программой по ссылке: https://chromedriver.chromium.org/ 
Программа запускается из консоли командой python3 main.py
Если нет желания при каждом использовании программы вводить пароль, его можно указать в файле password.txt после знака "равно"
При выполнении программы необходимо будет после вопроса программы ввести номер необходимой недели текущего месяца
По завершению работы программы в директории с программой будет создан файл фойл с названием finish_tabl.docx, этот файл и есть результат выполнения программы
